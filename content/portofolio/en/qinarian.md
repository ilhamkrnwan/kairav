# Qinarian — Tech Community Platform

A tech community platform built for innovators, where open-source collaboration meets cutting-edge digital solutions.

![Qinarian Platform](/portofolio/qinarian/preview.avif)

## Problem

Tech communities need a platform that feels modern, open, and collaborative. Most existing platforms are either too corporate or cluttered, lacking the welcoming environment developers need to collaborate and share knowledge effectively.

## Solution

Qinarian was built as a community hub prioritizing content and collaboration:

- **Project Showcase** — Clean project displays with detailed case studies and tech stack information
- **Community Hub** — A space for developers to connect, discuss, and collaborate
- **Resource Library** — Curated collection of tools, tutorials, and resources
- **Multi-Language Support** — Built-in i18n for global accessibility (Indonesian & English)
- **Content-Driven CMS** — Markdown-based for easy content updates
- **Modern UI/UX** — Clean interface that puts content first
- **SEO Optimized** — Built for discoverability and broad reach

## My Role

As **Fullstack Developer and UI/UX Designer**, I was responsible for:
- Architecting and developing the platform from scratch using Nuxt.js
- Implementing the i18n system for multi-language support
- Designing a clean, modern UI/UX with Tailwind CSS
- Setting up Markdown-based CMS (Nuxt Content) for content management
- SEO and performance optimization

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Nuxt.js, Vue 3, TypeScript |
| Styling | Tailwind CSS |
| CMS | Nuxt Content (Markdown-based) |
| i18n | @nuxtjs/i18n |
| Deployment | Vercel |

## Impact and Results

- A scalable, modern platform that brings tech communities together
- High performance with fast loading thanks to Nuxt.js SSR
- Clean design focused on content and collaboration
- Multi-language support for global reach
